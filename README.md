El objetivo del jugador sera el de avanzar sin  caerse al vacio o colisionar con las trampas de pinchos, recolectando una serie de monedas y powerUp para conseguir un mayor puntaje.

1. ganar doble de puntos
2. ocultar los pinchos del mapa para no colisionar
3. todas las plataformas se generan al mismo nivel de altura
4. combinar los power up anteriores en 1.
